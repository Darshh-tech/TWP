# Varenyam

Varenyam is a responsive luxury heritage-house website showcasing Banarasi craftsmanship, curated collections, master artisans, and private ownership enquiries.

## Features

- Responsive navigation for desktop and mobile
- Scroll-based content reveal animations
- Interactive product collection with detail modals
- Private enquiry form with a success modal
- Responsive layouts for desktop, tablet, and mobile screens

## Run locally

No installation or build step is required. Open `index.html` in a modern web browser.

Alternatively, serve the directory with any static-file server:

```bash
npx serve .
```

## Project structure

```text
.
├── Image/       # Product and collection images
├── index.html   # Page structure and content
├── script.js    # Navigation, modal, form, and reveal interactions
├── styles.css   # Site styling and responsive rules
└── TODO.md      # Project maintenance checklist
```

## Technology

The site uses plain HTML, CSS, and JavaScript with no runtime dependencies.
